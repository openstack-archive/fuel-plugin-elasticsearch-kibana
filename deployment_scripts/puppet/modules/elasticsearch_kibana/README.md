Elasticsearch_kibana module for Puppet
======================================

Description
-----------

Puppet module for configuring Elasticsearch server and kibana dashboard.

Usage
-----

```puppet
class {'elasticsearch_kibana':
  pv_name => '/dev/sdb',
}

```

Limitations
-----------

None.
