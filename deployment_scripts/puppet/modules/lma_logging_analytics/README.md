LMA Logging Analytics module for Puppet
=======================================

Description
-----------

Puppet module for configuring the Kibana dashboard and Elasticsearch.

Usage
-----

```puppet
class {'lma_logging_analytics::kibana': }
```

Limitations
-----------

None.
